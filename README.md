# 🎮 Rock Paper Scissors Game

A modern and interactive **Rock Paper Scissors** game built with **HTML, CSS, and JavaScript**, featuring smooth animations, sound effects, and dynamic visual feedback.

---

## 🌐 Live Demo

🔗 https://prerna-sharma-only.github.io/rock-paper-scissors/

---

## ✨ Highlights

* 🎯 Interactive gameplay with real-time results
* 🤖 Smart random computer opponent
* 🔊 Instant sound effects (optimized for low latency)
* 🌟 Winning animation with dynamic star background
* 🎥 Video-based visual effects for enhanced experience
* 🎨 Responsive UI with smooth color transitions
* 🔄 Reset & New Game functionality

---

## 🛠️ Tech Stack

| Technology | Purpose                    |
| ---------- | -------------------------- |
| HTML5      | Structure of the game      |
| CSS3       | Styling, animations, UI    |
| JavaScript | Game logic & interactivity |

---

## ⚙️ Key Features & Logic

### 🎮 Game Flow

* User selects Rock / Paper / Scissors
* Computer generates a random move
* Winner is determined using game logic
* Scores update instantly
* UI reacts with animations and sound

---

### 🔊 Audio Handling (Optimized)

* Preloaded audio for instant playback
* Controlled audio reset to prevent overlap
* Separate sound handling for win & lose states

---

### 🎥 Visual Experience

* Background transitions based on game result
* Star animation effect on winning
* Smooth video playback with timing control

---

## 🚀 Performance Improvements

* ⚡ Eliminated audio lag using preload technique
* 🧠 Managed concurrent animations and sound triggers
* 🎯 Reduced UI blocking with optimized event handling
* 🎥 Fixed video rendering delays using controlled playback
